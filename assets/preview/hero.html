Markdown that feels like ChatGPT

Use Typora for notes, research, specs, and technical writing while keeping the familiar ChatGPT Markdown rhythm: readable 16px text, generous line height, compact headings, soft borders, and calm surfaces.

The theme focuses on the Markdown body, not app chrome. The result is familiar, quiet, and practical for long documents.
