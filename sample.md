# ChatGPT Markdown, inside Typora

Typora ChatGPT mirrors the current ChatGPT web Markdown surface: measured spacing, clean system typography, rounded code cards, dotted links, citation pills, blockquotes, task lists, and responsive tables.

> The goal is not a loose “inspired by” palette. The Markdown body is tuned to feel like the official ChatGPT reading surface while staying native to Typora.

## Code

Inline code keeps a quiet gray pill: `const theme = "typora-chatgpt"`.

```javascript
const theme = "typora-chatgpt";
console.log("ChatGPT-style Markdown for Typora");
```
